this is a quick react-based about me page
# To get it started: 
* `git clone` repo URL
* `cd into directory`
* `npm install` # install the deps needed
* `npm start` # start up the nodeJS server locally via p3000


# Tools used:
  * react
  * react-dom
  * react-router
  * react MDL
  * react MDB
  * tachyons
  * font-awesome
  * draw.io
